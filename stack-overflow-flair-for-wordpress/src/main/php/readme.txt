Provides a widget for adding a Stack Overflow Flair badge to a WordPress blog.

== Description ==

The Stack Overflow Flair for WordPress plugin allows you to add your Stack Overflow user flair badge
to your WordPress blog.

== Installation ==

1. Upload the to your WordPress blog
2. Active the plugin
3. Configure your Stack Overflow user id and display name
4. Add the widget to your blog layout
